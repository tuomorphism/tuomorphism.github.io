# Tuomorphism personal site

Built with [AstroWind](https://github.com/arthelokyo/astrowind).

## Features

A nice personal portfolio and blog website with automatic content exporter from project repositories.

The content exporter (`tools/export_all.py`) is written with Python and has an associated docker image file.

## LLM Disclaimer

Majority of the front-end code is has been written in conjunction with ChatGPT-5. 
The content exporter has also been coded with ChatGPT-5 assistance.
